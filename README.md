# AI Resume Analyzer 🚀

## Overview
This project predicts candidate hiring probability using a hybrid ML approach.

## Features
- Data cleaning & preprocessing
- Feature engineering
- Logistic Regression model
- Threshold tuning for balanced classification

## Tech Stack
- Python
- Pandas, Scikit-learn
- XGBoost (experimentation)

## Results
- Balanced classification using threshold tuning
- F1 Score optimized at threshold = 0.4
